# Heart Failure Prediction System

## Objective
Predict the likelihood of heart failure using clinical data and compare multiple machine learning models.

## Features
- Data preprocessing and scaling
- Multiple models: Logistic Regression, KNN, Naive Bayes, Decision Tree, SVM
- Model evaluation using Accuracy, Confusion Matrix, ROC-AUC
- Visualization using Matplotlib and Seaborn

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Dataset
Heart Failure Clinical Records Dataset (300 records)
